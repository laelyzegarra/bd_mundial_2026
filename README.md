#Base de Datos Mundial 2026

Proyecto académico desarrollado en SQL Server que simula la gestión de un campeonato de fútbol (Mundial 2026).

---

#Descripción
Este proyecto consiste en el diseño e implementación de una base de datos relacional que permite almacenar y gestionar información sobre equipos, jugadores, partidos y fases del campeonato.

---

#Estructura de la Base de Datos
- Tabla de equipos
- Tabla de jugadores
- Tabla de partidos
- Tabla de fases
- Relaciones entre entidades

---

#Funcionalidades
- Creación de tablas y relaciones
- Inserción de datos
- Consultas SQL para análisis de información
- Organización de datos estructurados

---

#Tecnologías utilizadas
- SQL Server
- SQL Server Management Studio
